# 🏦 Bank App

A simple **Bank Application** built with **HTML, CSS, JavaScript, and Java** that simulates basic banking operations.

---

## 🚀 Features
- Deposit money 💰
- Withdraw money 💸
- Check current balance 📊
- Initial balance starts at **2000** 🏦
